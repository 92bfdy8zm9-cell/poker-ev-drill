# ポットオッズ & EV ドリル

ポーカーのコール判断（必要勝率・EV・アウツ→勝率）を素早く計算する反射練習ツール。

**▶ 遊ぶ:** https://92bfdy8zm9-cell.github.io/poker-ev-drill/

ランダムに実戦シナリオが出題され、数値を入力すると答え合わせと手計算の手順が表示されます。連続正解数・正答率を記録。スマホのブラウザでもそのまま動きます。

## モード
- **必要勝率（ポットオッズ）** — ポットとベットからコールに必要な最低勝率
- **コールのEV** — 読んだ勝率からコールの期待値がプラスかマイナスか
- **アウツ→勝率** — 2/4ルールでドロー完成の概算勝率
- **ミックス** — 上記をランダム出題
